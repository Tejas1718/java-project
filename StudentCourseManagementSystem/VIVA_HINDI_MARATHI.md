# Hindi / Marathi Viva Version

## Hindi Version

### 1. Project ka naam kya hai?
Mere project ka naam Student Course & Performance Management System hai.

### 2. Yeh project kis purpose ke liye banaya gaya hai?
Yeh project student details, course details, marks, attendance aur grades ko manage karne ke liye banaya gaya hai.

### 3. Is project me kaunsi programming language use hui hai?
Is project me Java programming language use hui hai.

### 4. GUI kis technology se banaya gaya hai?
GUI Java Swing se banaya gaya hai.

### 5. Data kaise store hota hai?
Data CSV files me local system par store hota hai.

### 6. Project ke main modules kaun se hain?
Student Management, Course Management, Enrollment Management aur Dashboard.

### 7. Grade kaise calculate hota hai?
Student ke marks ke basis par automatically grade generate hota hai.

### 8. Dashboard me kya dikhaya jata hai?
Dashboard me total students, total courses, total enrollments, average marks, average attendance aur top performers dikhaye jate hain.

### 9. Service layer ka kya kaam hai?
Service layer validation aur business logic handle karta hai.

### 10. Future me isme kya improvements kar sakte hain?
Login system, database, charts, search aur report export add kar sakte hain.
