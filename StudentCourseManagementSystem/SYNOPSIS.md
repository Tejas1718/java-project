# Project Synopsis

## Project Title

Student Course & Performance Management System

## Introduction

The Student Course & Performance Management System is a Java-based desktop application developed to manage academic records in a simple and organized way. It helps in maintaining student details, course information, enrollments, marks, attendance, and grades using a graphical user interface.

This project is developed as a college-level mini project to demonstrate Java programming, Swing-based GUI design, file handling, and object-oriented concepts.

## Need for the Project

In many cases, academic records are maintained manually, which can be time-consuming and difficult to organize. This project provides a digital solution to maintain records more effectively and reduce manual effort.

## Objectives

- To manage student details in digital form
- To manage course details efficiently
- To record student enrollments
- To track marks and attendance
- To calculate grades automatically
- To present academic statistics in dashboard form

## Proposed System

The proposed system is a desktop application developed in Java Swing. It contains separate modules for students, courses, and enrollments. It also includes a dashboard that gives performance-related insights. The data is stored locally in CSV files, making the system simple and lightweight.

## Functional Requirements

- Add new student details
- Update existing student details
- Delete student details
- Add, update, and delete course details
- Enroll students into courses
- Record marks and attendance
- Generate grades automatically
- Show dashboard summary

## Non-Functional Requirements

- Easy to use
- Lightweight
- Fast response
- Simple interface
- Low system requirement

## Technologies Used

- Java
- Java Swing
- CSV file storage
- OOP concepts
- Collections Framework

## Output of the Project

The system provides:

- Student records table
- Course records table
- Enrollment records table
- Dashboard with statistics
- Top performers summary

## Advantages

- Easy to operate
- Saves time
- Improves record organization
- Good for learning Java concepts
- Useful for academic presentations

## Limitations

- No database support
- No login authentication
- Works only on local machine

## Future Scope

- Database integration
- Search functionality
- Export reports
- Login system
- Better analytics and graphs

## Conclusion

The Student Course & Performance Management System is a useful Java mini project that demonstrates the implementation of GUI design, record management, file handling, and object-oriented programming in one application. It is suitable for academic submission and future enhancement.
