# Teacher Explanation in Simple English

Good morning sir/madam.

My project title is **Student Course & Performance Management System**.

This is a Java desktop application made using **Java Swing**. The purpose of this project is to manage academic records in a simple digital form. In this project, we can store student details, course details, enrollment details, marks, attendance, and grades.

The project has four main parts:

- Student Management
- Course Management
- Enrollment Management
- Dashboard

In the Student Management section, we can add, update, delete, and view student records.  
In the Course Management section, we can manage course information.  
In the Enrollment section, we can assign students to courses and store marks and attendance.  
The system automatically calculates grades based on marks.

The Dashboard shows important information such as:

- Total students
- Total courses
- Total enrollments
- Average marks
- Average attendance
- Top performers

I used **Java collections** to store records while the program is running, and I used **CSV files** to save the data locally. I divided the project into packages like model, service, persistence, and UI so that the code remains organized and easy to understand.

This project is useful because it reduces manual work and shows how Java can be used to build a complete desktop application with GUI, file handling, and object-oriented programming concepts.

In the future, this project can be improved by adding a database, login system, search feature, and report export.

Thank you.
